HTML + CSS Quick Cheat Sheet

Essential HTML Topics

Lists
  • ul, ol, li
  • dl, dt, dd
Phrasing/Inline
  • em, strong, small, mark
  • abbr, q, cite, code, kbd
  • sup, sub, span
Tables
  • table, caption
  • thead, tbody, tfoot
  • tr, th, td, colgroup, col
Forms
  • form, label, input, textarea, select, option, fieldset, legend, button
Media
  • img, audio, video

Must‑Know HTML Attributes

Global
  • id, class, title, style
  • hidden, tabindex
Links/Media
  • href, target (links)
  • src, alt (img)
  • controls, autoplay, loop, muted, poster (audio/video)
  • width, height, loading="lazy" (img)
Tables
  • colspan, rowspan
  • scope on th (e.g. col, row)
  • headers (a11y)
Form (tag)
  • action, method, enctype
  • novalidate, autocomplete
Inputs
  • type, name, value
  • placeholder, required, readonly, disabled
  • min, max, step, minlength, maxlength, pattern
  • checked (radio/checkbox), multiple (file/select), accept (file)
  • for on label → input id

Core CSS Properties

Text/Font
  • color, font-size, font-weight, font-style
  • text-align, text-decoration, line-height
Box Model
  • margin, padding, border, box-sizing
Backgrounds
  • background-color, background-image
  • background-size, background-repeat, background-position
Display/Layout
  • display (block/inline/inline-block/flex/grid)
  • position (static/relative/absolute/fixed/sticky)
  • top/right/bottom/left, z-index
Sizing/Other
  • width, height, max-width
  • overflow, opacity, cursor

Tip: Tables me th scope, Forms me label for ↔ input id, Media me alt/controls na bhoolna!
